== Description ==

User Table is a shortcode that shwos all the users in your WordPress site on the front-end 
and only accessible by administrators. The plugin does use DataTables dependency and 
displays 10 items per page. You can filter through search and by user role.


= Installation =

Once you have installed the plugin, you just need to activate it to enable it.


= Configuration =

There are no plugin options.

== Installation ==

1. Unzip the downloaded zip file.
2. Upload the plugin folder into the `wp-content/plugins/` directory of your WordPress site.
3. Activate `User Table` from Plugins page
